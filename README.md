# E-Commerce Sales Analysis & Customer Behavior Patterns – Data Cleaning Project

## Overview
This project focuses on cleaning and preparing an e-commerce sales dataset containing 1,200 customer transactions. The goal was to improve data quality and make the dataset ready for analysis and visualization.

## Dataset
- 1,200 records
- 14 columns
- Includes order details, customer information, payment methods, coupons, shipping, and sales data

## Data Cleaning Tasks
- Handled missing values in the `CouponCode` column
- Checked for duplicate records
- Validated data types
- Standardized column names and ttting
- Improved overall data consistency

## Key Findings
- 309 missing values were identified in `CouponCode`
- Missing values were replaced with "No Coupon"
- No duplicate records were found
- Dataset was successfully cleaned and prepared for analysis

## Tools Used
- Microsoft Excel
- Data Validation
- Data Cleaning Techniques

## Outcome
The final dataset is analysis-ready and can be used for:
- Sales analysis
- Customer behavior analysis
- Product performance evaluation
- Dashboard creation in Excel, Power BI, or Tableau
